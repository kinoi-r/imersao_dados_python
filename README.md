# imersao_dados_python
Este repositório reúne todos os notebooks e arquivos desenvolvidos durante a Imersão em Dados com Python da Alura, utilizando Google Colab e a biblioteca Pandas como principais ferramentas.

O conteúdo está organizado de acordo com as aulas da imersão, cobrindo desde os primeiros passos em Python para dados até a construção de um dashboard interativo.

Ao longo das aulas, são abordados os seguintes tópicos:

Exploração de dados com Pandas, entendendo a estrutura dos datasets e realizando análises iniciais

Limpeza e preparação dos dados, incluindo tratamento de valores ausentes e organização das informações

Criação de gráficos e storytelling com dados, transformando análises em insights visuais

Construção de um dashboard interativo, reunindo visualizações para comunicação dos resultados

Este repositório tem como objetivo documentar meu processo de aprendizado em análise de dados, servindo como registro dos estudos realizados durante a imersão e como material de consulta para projetos futuros em Python e dados.
